# Сравниваем вакансии программистов

Данный проект помогает вывести статистику о зарплате самых популярных языков программирования, 
на основе API HeadHunter и SuperJob.

## Как установить

Python3 должен быть уже установлен. Затем используйте pip 
(или pip3, если есть конфликт с Python2) для установки зависимостей:

```pip install -r requirements.txt```

Далее необходимо создать файл с разрешением .env и внести в него параметр:

    - SECRET_KEY_SJ=ваш токен 

Для того чтобы получить токен, необходимо пройти регистрацию продукта на сайте 
[SuperJob](https://api.superjob.ru/register).

## Как запустить скрипт

Для запуска необходимо прописать в консоль команду:

    ```python main.py```
